## aria2c<br>
Aria2 Service Bash<br>
Aria2服务脚本<br>
File storage directory<br>
文件存放目录<br>
<pre>
/etc/init.d/
</pre>
Support Start、Stop、Status、Restart<br>
支持启动、停止、状态、重启<br>
Usage<br>
用法：<br>
<pre>
service aria2c start
</pre>
Please Execute Before Use：<br>
使用前请执行：<br>
<pre>
chmod +x aria2c
</pre>
